2nd generation of AMD RDNA architecture, code name: Navi 2X.<br/>
10.3 generation of AMD GFX IP.

## Examples

**Discrete**:
* N6 fab: RX 6400 - 6500
* N7 fab: RX 6600 - 6900

**Integrated**:
* Radeon 680M
* Ryzen 6xxx / 7x20 / 7x35 / 7x45
* PS5 (36 CU)
* SteamDeck (8 CU)
